# AJRMart Web Application

**Developed By:** AJR Groups ([https://ajrmart.com](https://ajrmart.com))
**Client:** Pavan

---

## Project Overview

This project is a web application developed by **AJR Groups (ajrmart.com)** for the client **Pavan**. The repository contains all required files to run the application locally and deploy it to a production server.

---

## Prerequisites

* Node.js (v18 or later recommended)
* npm (comes with Node.js)

---

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd <project-folder>
```

2. Install dependencies:

```bash
npm install
```

---

## Environment Setup

Create a `.env` or `.env.local` file in the root directory and add required environment variables (example):

```env
PORT=3000
```

---

## Run the Application

Start the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```

---

## Build for Production

```bash
npm run build
npm start
```

---

## Deployment

The application can be deployed on any Node.js supported server such as:

* VPS / Dedicated Server
* AWS / Azure / GCP
* Vercel / Netlify

Make sure to:

* Set environment variables on the server
* Enable HTTPS
* Use PM2 or similar process manager for production
